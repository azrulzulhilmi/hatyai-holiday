# Hat Yai Holiday 2026 · Interactive Travel Map

An interactive travel guide and schedule visualizer for the Hat Yai holiday trip. Built using **Leaflet.js**, **Lucide Icons**, and styled with a custom high-fidelity dark frame layout inspired by the Linear design system.

## 🗺️ Core Features

* **High-Legibility Light Map**: Uses CartoDB Positron white/light tiles for clear visibility of streets, routes, and points of interest.
* **Numbered Route Sequence**: Map pins show numbers (`1`, `2`, `3`...) indicating the chronological order of destinations. The hotel basecamp is assigned the number **`0`** to act as a start and return anchor.
* **Unconnected Recommendation Pins**: Plotted custom dining recommendations (such as Ayutthaya Halal Beef Noodle, Marada Halal Restaurant, and Mue Chao Breakfast Halal) as standalone, solid-colored markers on the map without connecting lines.
* **Guidebook Popups**: Clickable map markers open popup cards containing real photography, clock indicators, and external **TikTok Guide** links.
* **English Day Filters**: Toggle itinerary paths day-by-day (All Days, Friday, Saturday, Sunday, Monday).
* **Detailed Timeline**: Scrollable list displaying mono-styled timestamps, category icons, venue thumbnail pictures, and calculated path distance indicators between consecutive travel stops.

## 🛠️ Technology Stack
* **HTML5**: Semantic document skeleton.
* **Vanilla CSS3**: Layout variables, custom surface ladder (`#010102` canvas, surface lifts), and responsive media query breakpoints.
* **JavaScript**: Map controls, Leaflet integration, Lucide icon builder, and UI click events.
* **Leaflet.js**: Free open-source interactive mapping.
* **Lucide Icons**: Clean SVG icon system.
